# Mini Food Admin Panel

**Egasi:** Muxtorv Muhammadrizo

Admin panel fast food restorani uchun. Menu, Buyurtmalar, Mijozlar, Xodimlar va Statistika boshqarishlari.

## Texnologiyalar
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP 7.4+
- **Database:** MySQL 5.7+

## Funksiyalar
✅ Menu boshqarish (taomlarni qo'shish/o'zgartirish/o'chirish)
✅ Buyurtmalarni boshqarish
✅ Mijozlarni boshqarish
✅ Xodimlarni boshqarish
✅ Statistika va Hisobotlar
✅ Sozlamalar

## O'rnatish
1. Repository klonlash: `git clone`
2. Database yaratish: `database/mini_food.sql` ni import qilish
3. Konfiguratsiyani sozlash: `backend/config.php`
4. Server ishlatish
